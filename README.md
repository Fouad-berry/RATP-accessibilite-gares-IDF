# Analyse de l’accessibilité des gares en Île-de-France

Ce projet analyse l’accessibilité des gares franciliennes à partir de données ouvertes (CSV, GEOJSON) et propose des visualisations et indicateurs pour Power BI.

## Structure du projet
- `data/raw/` : Données brutes (CSV, GEOJSON)
- `data/processed/` : Données nettoyées
- `notebooks/` : Notebooks d’analyse
- `scripts/` : Scripts Python
- `powerbi/` : Dashboard Power BI
- `images/` : Exports d’images

## Conseils d'installation

Il est recommandé d'utiliser un environnement virtuel Python pour isoler les dépendances du projet :

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
